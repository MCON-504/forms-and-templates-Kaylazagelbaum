# Forms and Templates

This repo contains demos, in-class exercises, and homework for learning Flask forms and Jinja2 templates.

## Start Here

- In-class exercise instructions: `exercises/EXERCISE_README.md`
- Homework instructions: `homework/HOMEWORK_README.md`

## Run the App

From the repo root, run the main Flask app with:

```bash
python app.py
```

If you are working on the exercises or homework, run the corresponding file instead:

```bash
python exercises/ninja_exercise.py
python homework/homework.py
```

## What is Where

- `demo/` - Reference examples for Jinja2 templates
- `exercises/` - In-class work and templates for practice
- `homework/` - Homework app and templates

If you were looking for `HOMEWORK_EXERCISE.md`, the current homework instructions live in `homework/HOMEWORK_README.md`.
